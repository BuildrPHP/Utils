# Installation

## With composer

Run the following command in your project root directory:

```bash
$ composer require buildr/utils:1.0.*
```

Or, simply add this project to your `composer.json` and you good to go.

```json
{
  "require": {
    "buildr/utils": "1.0.*"
  }
}
```
